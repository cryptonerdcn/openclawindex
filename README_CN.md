# OpenClawIndex

OpenClaw 变种与生态项目的一站式多语言目录。

[English README](./README.md)

## 网站定位

OpenClawIndex 用于快速发现和比较 OpenClaw 相关项目，包括：

- 各类变种实现（如不同语言重写）
- 相关生态服务（工具、平台、分发、集成）
- 每个项目的核心信息、链接与多语言说明

目标是在一个页面内完成发现、筛选、对比和跳转。

## 用户可感知特征

- 单页面目录体验：减少来回跳转。
- 多语言切换：`EN / 中 / 日` 快速切换且上下文不丢失。
- Facts / Opinions 双层阅读：先事实，后观点。
- 统一条目结构：横向比较更直接。
- 高可读筛选检索：按分类和关键词快速定位。

## 如何贡献

通过 GitHub Issue 提交收录申请：

1. `Submit Claw Entry`（推荐）：结构化表单，适合完整提交。
2. `Quick Project Suggestion`：快速提名，适合先提供线索。

Issue 模板目录：`.github/ISSUE_TEMPLATE/`  
详细说明：`CONTRIBUTING.md`
